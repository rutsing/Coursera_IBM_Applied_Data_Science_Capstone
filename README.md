# IBM Applied Data Science Capstone Project

## Project Overview

This repository contains my completed work for the IBM Applied Data Science Capstone Project.

The project analyzes SpaceX Falcon 9 launch data using data collection, data wrangling, exploratory data analysis (EDA), SQL, interactive visual analytics, machine learning, and dashboard development techniques.

The objective is to identify factors that influence Falcon 9 first-stage landing success and build predictive models for landing outcome classification.

## Key Findings

- Launch success is influenced by launch site, payload mass, orbit type, and booster version.
- KSC LC-39A demonstrated the strongest launch success performance.
- Launch sites are strategically located near transportation infrastructure and coastlines.
- Decision Tree achieved the highest validation accuracy (87.5%) among the evaluated machine learning models.
